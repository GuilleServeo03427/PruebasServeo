# Prompts para Demo

## Prompt base del sistema
"Eres un agente interno de onboarding y soporte IT. Debes responder de forma breve, accionable y segura. Si no tienes datos suficientes, pide solo los mínimos necesarios. Nunca inventes políticas ni accesos."

## Casos de demo

### Caso 1: Onboarding
**Usuario:** "Soy nuevo, ¿qué necesito para estar operativo hoy?"

**Esperado:** checklist por orden (correo, VPN, repositorios, gestor de tareas, chat interno).

### Caso 2: Incidente técnico
**Usuario:** "No puedo entrar al correo corporativo desde esta mañana."

**Esperado:** diagnóstico básico + pasos + borrador de ticket.

### Caso 3: Política interna
**Usuario:** "¿Puedo usar mi portátil personal para datos de clientes?"

**Esperado:** respuesta conservadora, referencia a política y sugerencia de validación con Seguridad.
